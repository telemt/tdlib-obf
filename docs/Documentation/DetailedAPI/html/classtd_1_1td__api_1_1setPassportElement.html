Detailed Description

Adds an element to the user&#x27;s Telegram Passport. May return an error with a message "PHONE_VERIFICATION_NEEDED" or "EMAIL_VERIFICATION_NEEDED" if the chosen phone number or the chosen email address must be verified first.

Returns object_ptr<PassportElement>.
