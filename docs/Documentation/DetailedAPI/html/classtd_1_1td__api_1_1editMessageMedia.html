Detailed Description

Edits the media content of a message, including message caption. If only the caption needs to be edited, use editMessageCaption instead. The type of message content in an album can&#x27;t be changed with exception of replacing a photo with a video or vice versa. Returns the edited message after the edit is completed on the server side.

Returns object_ptr<Message>.
