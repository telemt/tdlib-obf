Detailed Description

Invites a bot to a chat (if it is not yet a member) and sends it the /start command; requires can_invite_users member right. Bots can&#x27;t be invited to a private chat other than the chat with the bot. Bots can&#x27;t be invited to channels (although they can be added as admins) and secret chats. Returns the sent message.

Returns object_ptr<Message>.
